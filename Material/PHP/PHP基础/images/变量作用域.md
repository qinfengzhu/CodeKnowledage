### 变量的作用域

1. `超全局变量`

| 变量      | 说明                      |
| --------- | ------------------------- |
| $_GET     | 地址栏GET提交             |
| $_POST    | 表单POST提交              |
| $_FILES   | 文件上传变量              |
| $_SESSION | 会话变量                  |
| $_COOKIE  | cookie值变量              |
| $_GLOBALS | 全局变量                  |
| $_REQUEST | 包含$_GET,$_POST,$_COOKIE |
| $_SERVER  | 服务器环境变量            |
